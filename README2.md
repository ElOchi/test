# simple_shell

## Installation
Clone the repository into a new directory
```bash
git clone https://github.com/csoriano2832/simple_shell.git
```
Compile:
```bash
gcc -Wall -Werror -Wextra -pedantic *.c -o simpe_shell
```
Execute:
```bash
./simpe_shell
```
